# Examples

This directory contains minimal example requests for One Answer.

Current files:
- `request.decision.json`
- `request.debug.json`
- `request.architecture.json`
- `request.plan.json`
- `request.strategy.json`
- `request.tradeoff.json`

Golden eval cases live under:
- `golden/*.eval.json`

Run the current local eval loop with:

```bash
npm run eval
```

These request files are the seed fixtures for MCP/API demos and golden-example tests.
